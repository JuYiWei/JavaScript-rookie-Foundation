2.4 闭包

  • 举例
  • 解释

对象拥有私有变量；

举例

    var add = (function() {
        var count = 0;
        return function () {
            console.log(count);
            return count+=1;
        }
    })();

    document.write(add());// 1
    document.write(add());// 2
    document.write(add());// 3

解释

1 add 作为变量，指向自调用函数，初始化 c，且自调用函数返回 普通函数；
2 经过自调用后 add = function () { return c+=1; }；
3 c 被 add 对象的函数持有，但又在自调用时初始化
4 c 就只能通过 add() 修改
