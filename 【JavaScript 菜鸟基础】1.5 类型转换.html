1.5 类型转换

  • 类型

  • constructor 属性
  • 字符串 <=> 数字
  • Boolean <=> 字符串
  • Boolean <=> 数字
  • 日期 <=> 字符串
  • 日期 <=> 数字
  • 不同类型运算处理

类型

  • 数据类型
string
number
boolean
object
function
  • 对象类型
Object
Date
Array
  • 其他
null
undefined

typeof 操作符

  • 检查变量类型
typeof "John"                // 返回 string 
typeof 3.14                  // 返回 number
typeof false                 // 返回 boolean
typeof [1,2,3,4]             // 返回 object
typeof {name:'John', age:34} // 返回 object
typeof function () {}         // 返回 function
typeof myCar                  // 返回 undefined (如果 myCar 没有声明)
typeof undefined              // 返回 undefined
typeof null                   // 返回 object

constructor 属性

  • 检查变量构造函数
document.write([1,2,3,4].constructor)   //function Array() { [native code] }

字符串 <=> 数字

  • 字符串 => 数字
    var str = '1.23';
    
    var a = Number(str);
    var b = parseInt(str);
    var c = parseFloat(str);
  • 数字 => 字符串
    var num = 1.23;
    
    var a = String(num);
    var b = num.toString();
    var c = num.toFixed(1);//指定小数位数
    var d = num.toPrecision(2);//指定有效长度

Boolean <=> 字符串

  • Boolean => 字符串
    var checked = true;
    
    var a = String(checked);
    var b = checked.toString();
  • 字符串 => Boolean
    var a = Boolean(true);

Boolean <=> 数字

  • Boolean => 数字
    var checked = true;

    var a = Number(checked);
  • 数字 => Boolean
    var a = Boolean(0);

日期 <=> 字符串

  • 日期 => 字符串
    var date = new Date();

    var a = String(date);
    var b = date.toString();
    var c = date.toUTCString();
  • 字符串 => 日期
    var s ='2017-04-18 09:16:15';
    var date = new Date(s );
    console.log(date.toLocaleString());

日期 <=> 数字

  • 日期 => 数字
    var date = new Date();
    
    var a = date.getTime();// 时间戳，毫秒数
    var year = date.getYear();// 年
    var month = date.getMonth();// 月
    var day = date.getDate();// 日
  • 数字 => 日期
    var date = new Date(1000);// 毫秒数
